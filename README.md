RBFS - RapiD's Blog From Scratch
===

```
Just a Toy

I'll do all `backend-things` from scratch
```

Requirements
---

* node 5+
* npm


Build
---

### 1. Install Tools

```
npm install typescript gulp-cli tsd -g
```

### 2. Install dependencies

```
npm install
```

### 3. Install typings

```
tsd install
```

### 4. Build

```
gulp compile
```
